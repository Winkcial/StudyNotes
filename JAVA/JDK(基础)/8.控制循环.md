# Java 控制循环

- 有 break 和 contiune
- if 、switch 、while 、do while 、for 、 for each

**if 语句**

```java
    if (i > 100) {
        System.out.println("输入的数字大于100");
    }else{
        System.out.println("输入的数字小于于100");
    }
```

**switch 语句**

```java
switch(表达式) {
    case 值1:
        语句块1;
        break;
    case 值2:
        语句块2;
        break;
    …
    case 值n:
        语句块n;
        break;
    default:
        语句块n+1;
        break;
}
```

**while 循环**

```java
while(条件表达式) {
    语句块;
}
```

**do while 循环**

```java
do {
    语句块;
}while(条件表达式);
```

**for 循环**

```java
for(条件表达式1;条件表达式2;条件表达式3) {
    语句块;
}
```

**for each 循环**

```java
for(类型 变量名:集合) {
    语句块;
}
```
