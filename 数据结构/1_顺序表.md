# 顺序表

顺序表在逻辑上是连续的，在物理上也是连续的，即一段地址连续的储存空间。

c 语言

```c
# define maxsize 100
typedef int elemtype;
typedef struct sqlist{
    int length;//顺序表长度
    int data[maxsize];//储存数据
}list;
list L;//定义一个顺序表
```

初始化一个顺序表

```c
list init(){
    list *L;
    L=(list *)malloc(sizeof(list));
    if(!L){
        printf("error !!!");
        exit(1);
    }
    return *L;
}
```
