# 组管理终端命令

>Tip:创建/删除组都需要使用 `sudo` 

## 创建组

| 命令                      | 作用                      |
| ------------------------- | ------------------------- |
| groupadd 组名             | 添加组                    |
| groupdel 组名             | 删除组                    |
| cat etc/group             | 确认组信息                |
| chgrp -R 组名 文件/目录名 | 递归修改文件/目录所属的组 |

>Tips:组的相关信息保存在 /etc/group 文件中

```
$ cat /etc/group # 查看 group 文件
```
