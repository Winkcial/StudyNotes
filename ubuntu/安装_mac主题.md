# 安装 mocOS 主题

## 安装 Tweaks 

```shell
sudo apt install gnome-tweak-tool
```

## 手动安装 Dash To Dock

下载文件

解压

重命名

移动到文件夹 /home/.local/share/gnome-shell/extensions


Alt+F2 再输入 r 重启桌面

终端输入 gnome-tweaks 打开 Tweaks


## 下载主题和图标

下载文件

移动文件夹

```shell
sudo mv xxxxx /usr/share/themes

sudo mv  xxxx /usr/share/icons
```

## 设置 Tweaks

更改按键位置

更改 Dock 位置，大小，自动隐藏

重启
