
## 操作文件

计算机中操作文件一般三步

- 打开文件
- 读或者写文件
- 关闭文件

## 操作文件的函数方法

在 Python 的内置函数中对于文件的操作需要记住一个函数三个方法

| 函数或方法 | 说明                         |
| ---------- | ---------------------------- |
| open       | 打开文件，返回文件的操作对象 |
| read       | 将文件内容读取到内存中       |
| write      | 将指定内容写入到文件中       |
| close      | 关闭文件                     |

- open 函数打开文件，返回文件对象
  -write. read, close 都需要文件对象
