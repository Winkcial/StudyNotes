# 文件打开方式

当用 `open()` 打开一个文件后，会返回一个文件对象

```py
f=open("文件地址","打开方式")
```

| 打开方式 | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| r        | 只读。文件不存在会抛出异常                                   |
| w        | 只写，文件不存在，创建文件                                   |
| a        | 已追加的方式打开文件，文件存在，文件指针会在文件的末尾       |
| r+       | 以读写的方式打开文件，文件不存在抛出异常                     |
| w+       | 以读写方式打开文件，文件不存在创建新文件                     |
| a+       | 以读写的方式打开文件，文件指针在文件的末尾，文件不存在，创建新文件 |
