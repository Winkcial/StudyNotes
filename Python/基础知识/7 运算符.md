
## 算数运算符

| 运算符 | 描述   |
| ------ | ------ |
| +      | 加     |
| -      | 减     |
| *      | 乘     |
| /      | 除     |
| //     | 取整   |
| %      | 取余数 |
| **     | 幂乘   |

> 在 Python 中 `*` 还可以用于字符串的幂乘，结果就是字符串重复指定的幂次

## 赋值运算符

| 运算符 | 实例                |
| ------ | ------------------- |
| =      | a=b+c 等效于 a=b+c  |
| +=     | a+=b 等效于 a=a+b   |
| -=     | a-=b 等效于 a=a-b   |
| *=     | a*=b 等效于 a=a*b   |
| /=     | a/=b 等效于 a=a/b   |
| //=    | a//= 等效于 a=a//b  |
| %=     | a%=b 等效于 a=a%b   |
| **=    | a**=b 等效于 a=a**b |
