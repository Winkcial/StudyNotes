# 处理 String

## 相关函数

| 函数| 功能|
|---|---|
|str.title()|字符串开头变大写
str.upper()|将字符串中字母全部变大写
str.lower|将字符串中字母全部变小写
str.str(number)|将一个数字转换成字符串
len(str)|获取字符串的长度
str.count(str_1)|统计字符串 str_1 在字符串 str 中出现的次数
str.index(字符串)|获取字符串在 str 中的位置



### 链接字符串

使用加号链接

```
a="1"
b="2"
c=a+" "+b
```
那么 c='12'

### 制表符和换行符

制表符：`\t`

换行符：`\n`

### 删除空白

使用 `rstrip()` 可以删除字符串中多余的空白

这种删除是暂时的，要想永久删除必须将操作结果保存到原变量中

`lstrip()` 删除字符串开头的空白
