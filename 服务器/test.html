换源

ubuntu 官方在中国设有服务节点，可以换成官方源

把 PC 上的源文件上传到服务器

  1. 打开你的 PC 终端，输入以下命令
cd /etc/apt
cp ./sources.list ~/sources.backlist
  1. 上传文件 sources.list 到服务器
  2. 在服务终端输入以下命令
sudo cp ./sources.backlist /etc/apt/sources.backlist
cd /etc/apt
sudo rm sources.list
sudo mv ./sources.backlist /etc/apt/sources.list
sudo apt-get update
