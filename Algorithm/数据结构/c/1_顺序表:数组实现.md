# 顺序表

顺序表在逻辑上是连续的，在物理上也是连续的，即一段地址连续的储存空间。

```c
typedef struct SqList{
    int length;//顺序表长度
    Elemtype data[maxsize];//储存数据
}SqList;
```

```c
SqList *L;//定义一个顺序表
L=(SqList*)malloc(sizeof(SqList));// 分配内存空间
```

### 基本操作

**初始化一个顺序表**

```c
SqList* init(){
    SqList *L;
    L=(SqList *)malloc(sizeof(SqList));
    if(!L){
        printf("error\n");
        exit(0);
    }
}
```
