# 顺序表

## 栈的顺序储存类型

```c
#define maxsize 50
typedef struct{
    Elemtype data[maxsize];//存放栈中元素
    int top;//栈指针
}sqstack;
```
初始时设置S.top=-1

## 一些基本操作

初始化

```c
void initstack(sqstack &S){
    S.top=-1;
}
```


进栈

```c
bool Push(sqstack &S,Elemtype x){
    if(S.top==maxsize-1){
        return false;
    }
    else{
        S.data[++S.top]=x;
    }
}
```

出栈
```c
bool Pop(sqstack &S,Elemtype &x){
    if(S.top==-1){
        return false;
    }
    else{
    x=S.data[S.top--];
    return true;
    }
}
```
